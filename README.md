## Dependencies

What you need to run this app:
* `node` and `npm` (Use [NVM](https://github.com/creationix/nvm) or [n](https://github.com/tj/n))
* Ensure you're running Node (`v14.0.x`+) and NPM (`v6.14.x`+)
